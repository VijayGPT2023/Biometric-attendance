Habitual late-comer definition (Top Management feedback 2026-04-10):
- Employee has MORE THAN 6 anomaly days in a month
- This happens for 3 CONSECUTIVE months
- Such employees flagged as "Habitual Late-comer"
- Visible to Group Head, DG, DDG levels
- This is a separate report/list, not just a filter

**Why:** Government rules require identifying habitual offenders for disciplinary action.

**How to apply:** Need to track anomaly counts per employee per month across sessions. When 3 consecutive months each have >6 anomalies, flag the employee.
